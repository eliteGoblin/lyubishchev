# Lyubishchev

Personal time and metric tracking system, following Lyubishchev's method, mentioned in his bioriphy [This strange life](https://sudonull.com/post/171201-Granin-This-strange-life)
## Setup

Setup venv:
```sh
sudo rm -rf .venv
python3 -m venv .venv
source .venv/bin/activate
which python
```

Run all CI scripts:

```sh
scripts/ci/ci
```
